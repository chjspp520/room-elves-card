一、整体介绍：
房间精灵（Room Elves Card）是一款高度可定制、性能优化到位的 Home Assistant 控制卡片，适合需要集中管理多设备、追求界面美观、注重使用体验的用户，实现一站式房间智能设备管理，内置多种主题【light(亮色)、dark(暗色)、black(纯黑)、darkgray(深灰)、transparent(半透明)】。通过简单的 YAML 配置，即可打造出功能强大且视觉统一的移动端智能家居控制面板。
